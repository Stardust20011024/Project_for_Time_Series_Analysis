# 项目简介
本项目基于 UCI 的 Appliances Energy Prediction 数据集，研究并预测独栋住宅中家用电器的能耗时间序列。不同于引入气象等外生变量的多变量模型，本研究专注于纯单变量时间序列建模，仅利用历史能耗数据评估其预测潜力。

项目采用 Box–Jenkins 方法论，包括数据预处理、平稳性检验以及 SARIMA 模型的识别与估计。同时对比了**静态预测**与**滚动预测**（Rolling Forecast）两种策略，结果表明滚动预测在中长期预测中具有更优的稳定性和准确性。

实验结果显示，所构建的动态模型在测试集上达到了 R^2 = 0.418，说明住宅能耗具有显著的内在周期性特征，可在不依赖复杂环境传感器的情况下实现有效建模与预测。
